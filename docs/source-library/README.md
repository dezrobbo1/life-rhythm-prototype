# Life Rhythm source library

This folder is for project evidence and design-source documentation. It is not loaded by the app runtime.

The app remains local-first and non-clinical.

Use these sources in order:

1. `../DOCUMENTATION_AUTHORITY.md` for the repository documentation hierarchy.
2. `../../app/docs/life-rhythm-current-design-spec.md` for current `/app` product direction and implementation status.
3. `../../docs/ux/Life_Rhythm_Design_Specification_v1_2_Evidence_Balanced.md` for evidence weighting and screen-level research traceability.
4. `SOURCE_INDEX.md` for the source archive and packet map.

The supplied `Life_Rhythm_All_Project_Sources_2026-07-12.zip` is the current provenance bundle. It contains the Design Source Pack v1.2, the Packet 1-25 source collection, project-source governance additions, and current visual review references. It is source material only and must not be imported by the production app.
